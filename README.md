# PEREZ-BEGINO-REPO

# Bullet Hell Game

## Description

A fast-paced bullet hell game where the player dodges enemy attacks and survives multiple waves. The game features a final boss (Kitsune) with unique mechanics and phase-based behavior.

---

## Features

- Multiple waves with increasing difficulty
- Unique Wave 10 mechanic (auto-attack mode)
- Apex boss encounter with enhanced difficulty and mechanics
- Kitsune boss with multiple phases
- Different bullet patterns and attacks
- Power-up system with temporary and permanent effects
- In-game shop for upgrades and stat boosts

---

## How to Run

1. Clone the repository:
   git clone <https://github.com/Soliquifide/PEREZ-BEGINO-REPO.git>

2. Open the project in VS Code

3. Make sure Java (JDK 8 or higher) is installed

4. Run:
   BulletHellGame.java

---

## Controls

- Move: WASD
- Shoot: Mouse Click / Space

---

## Gameplay

- Survive waves 1–9 with manual control
- Wave 10 switches to auto-attack mode
- Defeat the Kitsune boss to win
- Boss has multiple phases with different attack patterns

---

## Important Notes

- Make sure all `.wav` files are inside the project folder
- Do not rename or move audio files or music will not play
- Ensure all project files are properly cloned before running
